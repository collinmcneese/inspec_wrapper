# inspec_wrapper

Inspec wrapper package which will load additional controls based upon OS type and run using Chef Habitat under the Effortless pattern.

```
inspec_wrapper
├── README.md
├── chefignore
├── controls
│   └── wrapper.rb
├── habitat
│   ├── default.toml
│   ├── plan.ps1
│   └── plan.sh
├── inspec.yml
├── libraries
```
